# 🏪 Franchise Performance Analysis

## Project Overview
Multi-store business analysis of a NYC coffee shop 
franchise with 3 locations using Python.

## Business Questions Answered
- Which store generates the most revenue?
- What is each store's best selling product?
- What is the average daily revenue per store?

## Key Findings
- 🥇 Hell's Kitchen: $236,511 total revenue
- 🥈 Astoria: $232,243 total revenue  
- 🥉 Lower Manhattan: $230,057 total revenue
- Each store has different best product — local taste matters!
- Average daily difference only $35 — healthy franchise!

## Tools Used
- Python, Pandas, Matplotlib
- Google Colab
- Dataset: 149,116 transactions
